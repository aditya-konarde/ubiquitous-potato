# Side Projects

A small monorepo for experimental side projects. These are useful, personal tools rather than polished products.

## Warning

All projects in this repository should be considered alpha-quality software. They may break, behave unexpectedly, corrupt or delete data, interfere with browser sessions, expose rough edges, or stop working without notice.

Use everything here at your own risk. No warranties, guarantees, support, maintenance commitments, or fitness for any purpose are provided.

## Projects

| Project | What it does | Status | License |
| --- | --- | --- | --- |
| [`amdgpu-exporter`](./amdgpu-exporter/) | Prometheus exporter for AMD GPU telemetry from Linux sysfs. | Functional, best documented project in this repo. | MIT, see `amdgpu-exporter/LICENSE` |
| [`younope`](./younope/) | Chrome extension that blocks selected YouTube channels and hides their videos. | Experimental browser extension. | GPLv3 via root `LICENSE` |
| [`redirectit`](./redirectit/) | Minimal Chrome extension that redirects `www.reddit.com` to `old.reddit.com`. | Tiny utility extension. | GPLv3 via root `LICENSE` |

## Repository layout

```text
amdgpu-exporter/  Rust Prometheus exporter plus Grafana dashboard
younope/          Manifest V3 YouTube channel blocker extension
redirectit/       Manifest V3 Reddit redirect extension
```

## Development checks

### amdgpu-exporter

```bash
cd amdgpu-exporter
cargo fmt -- --check
cargo clippy -- -D warnings
cargo test
cargo build --release
```

### Browser extensions

Validate JSON and JavaScript syntax from the repository root:

```bash
python3 -m json.tool younope/manifest.json >/dev/null
python3 -m json.tool redirectit/manifest.json >/dev/null
python3 -m json.tool redirectit/rules.json >/dev/null
node --check younope/content.js
node --check younope/popup.js
node --check younope/options.js
```

## Notes

- Browser extension `_metadata/` directories are generated by Chromium and are intentionally ignored.
- Build outputs such as Rust `target/` directories are intentionally ignored.
- This repo intentionally has mixed licensing. Check each project before reuse.
